Professioneel vakmanschap



Mijn voortgang

Ik heb tijdens mijn vorige op.leiding al 3 keer stage moeten lopen. Dit heb ik gedaan bij een multinational en bij kleinere bedrijven. Tijdens het themasemester Mobile Development heb ik een project gedaan voor Transavia op het hoofdkantoor.
Hierdoor heb ik al ervaring in het werken binnen professionele bedrijven en weet ik mij in deze omgeving ook professioneel op te stellen.

Hoe ga ik deze competentie verder ontwikkelen?

Hoewel ik eerder stage heb gelopen, heb ik nog niet veel ervaring met werken met ervaren programmeurs. Hierdoor denk ik dat de kwaliteit/professionaliteit van mijn code nog niet goed genoeg is om te voldoen aan bedrijfstandaarden. Dit hoop ik te verbeteren bij mijn stagebedrijf.
